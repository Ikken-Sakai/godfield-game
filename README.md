# 🎮 GodField Clone - AIパワード運ゲーカードバトル

ゴッドフィールドを参考にしたWebブラウザ上で動作するカードゲームです。

## 📋 概要

- **ゲーム形式**: ターン制カードバトル
- **プレイヤー**: 1人（vs CPU）または 2人対戦
- **目的**: 相手のHPを0にして勝利

## 🎯 特徴

- ブラウザ上で動作（Google Chrome推奨）
- シンプルで直感的なUI
- 将来的にAIによる自動カード生成機能を追加予定

## 🚀 開発フェーズ

| フェーズ | 内容 | 状態 |
|---------|------|------|
| Phase 1 | ゴッドフィールドの基本再現 | 🔄 進行中 |
| Phase 2 | AI自動カード生成機能 | ⏳ 予定 |
| Phase 3 | パラメータ未知化（運ゲー強化） | ⏳ 予定 |

## 🛠️ 技術スタック

- **フロントエンド**: HTML5 / CSS3 / JavaScript
- **動作環境**: Ubuntu 20.04 / Google Chrome

## 🎮 遊び方

1. `index.html` をブラウザで開く
2. ゲーム開始ボタンをクリック
3. 手札からカードを選んで攻撃・防御
4. 相手のHPを0にすれば勝利！
